# HybrisTestTask

To start app you need to run start.bat file (Hybris-1.0.jar and start.bat must be in the same directory)
or from command line change directory to the root of the project, where located Hybris-1.0.jar,
and execute following command: java -jar Hybris-1.0.jar

To run test in the root of project execute(add credential for db from google form):
mvn test
